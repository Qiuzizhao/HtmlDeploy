# 🚀 HtmlDeploy

✨ **HtmlDeploy** 是一个轻量、极速的静态网页部署与管理平台！专为快速发布和预览 HTML 静态项目（如小游戏、静态网页作业等）而设计。

## 🛠️ 技术栈 (Tech Stack)
- **后端 (Backend)**: 🟢 Node.js + Express
- **数据库 (Database)**: 🗄️ SQLite (轻量级本地存储)
- **前端 (Frontend)**: 🌐 原生 HTML / CSS / JavaScript (零构建、极速加载)
- **进程守护 (Process)**: ⚙️ PM2
- **部署代理 (Proxy)**: 🛡️ Nginx

## 🌟 核心功能 (Features)
- **📁 一键上传部署**: 无需复杂配置，ZIP 压缩包或单 HTML 文件一键上传并在线解压部署。
- **🏫 班级分类管理**: 支持创建不同班级/分类，支持拖拽排序，方便整理不同批次的项目。
- **🔒 密码访问控制**: 支持全局后台密码，以及针对单个班级设置独立的访问密码。
- **🎮 极致预览体验**: 独立的卡片式项目预览，专门为 WebGL/Canvas 等 HTML5 小游戏进行底层渲染优化（开启 GPU 硬件加速，拒绝掉帧卡顿）。
- **☁️ GitHub 自动备份**: 底层集成 Git 自动化脚本，所有上传的网页和数据变更都会自动、静默地 Push 到 GitHub 仓库，数据永不丢失！
- **✨ 现代 UI 设计**: 采用明亮清新的色彩搭配和毛玻璃（Glassmorphism）视觉风格，支持响应式布局（适配 PC、平板、手机端）。

## 📝 其他亮点 (Highlights)
- 序号生成绝对递增，项目删除不复用旧序号。
- 后台列表数据采用无感局部刷新（静默刷新），提供丝滑的后台操作体验。
- 自动提取网页作者、并提供前台快捷检索与筛选。
